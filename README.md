# calculator


# Introduction

This project focuses on building a basic calculator using front-end web technologies.
The goal is to create a functional calculator that can handle user input and perform real-time calculations.

# What This Project Demonstrates

By completing this project, I will have practiced and demonstrated:

-Writing clean and structured HTML

-Creating a responsive and user-friendly UI with CSS

-Implementing calculator logic using JavaScript

-Handling user input and updating the display dynamically

-Organizing files into a clear project structure

-Using Git and GitHub for version control

-Building a strong foundation for future JavaScript projects
